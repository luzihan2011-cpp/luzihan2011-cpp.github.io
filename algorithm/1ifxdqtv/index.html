KM 算法

匈牙利算法又称为 KM 算法，可以在 O(n3)O(n^3)O(n3) 时间内求出二分图的 最大权完美匹配。

注

KM 算法只能解决两边点数相同时的最大匹配问题。

考虑到二分图中两个集合中的点并不总是相同，为了能应用 KM 算法解决二分图的最大权匹配，需要先作如下处理：将两个集合中点数比较少的补点，使得两边点数相同，再将不存在的边权重设为 000，这种情况下，问题就转换成求最大权完美匹配问题，从而能应用 KM 算法求解。

定义

下面作出如下定义：

权重： w(u,v)w(u,v)w(u,v) 表示点 uuu 到点 vvv 的边权

可行顶标： 给每个节点 iii 分配一个权值 l(i)l(i)l(i)，对于所有边 (u,v)(u,v)(u,v) 满足 w(u,v)≤l(u)+l(v)w(u,v) \leq l(u) + l(v)w(u,v)≤l(u)+l(v)。

相等子图： 在一组可行顶标下原图的子图，包含所有点但只包含满足 w(u,v)=l(u)+l(v)w(u,v) = l(u) + l(v)w(u,v)=l(u)+l(v) 的边 (u,v)(u,v)(u,v)。

定理 1: 对于某组可行顶标，如果其相等子图存在完美匹配，那么，该完美匹配就是原二分图的最大权完美匹配。

有了定理 1，我们的目标就是通过不断的调整顶标，使得相等子图是完美匹配。

如何调整顶标？

因为两边点数相等，所以假设两边点数都为 nnn，lx(i)lx(i)lx(i) 表示左边第 iii 个点的顶标，ly(i)ly(i)ly(i) 表示右边第 iii 个点的顶标，w(u,v)w(u,v)w(u,v) 表示左边第 uuu 个点和右边第 vvv 个点之间的权重。

首先初始化一组可行顶标，例如：
左边都初始化为与它相接的边的边权最大值，右边都设为0

lx(i)=max⁡1≤j≤n{w(i,j)}, ly(i)=0lx(i) = \max_{1\leq j\leq n} \{ w(i, j)\},\, ly(i) = 0 lx(i)=1≤j≤nmax​{w(i,j)},ly(i)=0

然后选一个未匹配点，如同最大匹配一样求增广路。找到增广路就增广，否则，会得到一个交错树。

提示

交错树

从未匹配点 r 进行 DFS 或 BFS 寻找增广路的过程中产生的树称为交错树，r 是交错树的根。设 T=(Vt,Et)T=(V_t,E_t)T=(Vt​,Et​) 为再寻找增广路时产生的交错树。

定义：

  • 偶点（黑点）为树上深度为偶数的点。
  • 奇点（白点）为树上深度为奇数的点。

所以 aaa 值的选择，显然得是 S-T' 当中最小的边权，即

a=min⁡{lx(u)+ly(v)−w(u,v)∣u∈S,v∈T′}a = \min \{ lx(u) + ly(v) - w(u,v) | u\in{S} , v\in{T'} \} a=min{lx(u)+ly(v)−w(u,v)∣u∈S,v∈T′}

当一条新的边 (u,v)(u,v)(u,v) 加入相等子图后有两种情况

  • vvv 是未匹配点，则找到增广路
  • vvv 和 S′S'S′ 中的点已经匹配

每次修改顶标的时候，交错树中的边不会离开相等子图，那么我们直接维护这棵交错树。

维护方式

我们对 TTT 中的每个点 vvv 维护

设 slack(v)=min⁡{lx(u)+ly(v)−w(u,v)∣u∈S}\text{设}\ slack(v) = \min \{ lx(u) + ly(v) - w(u,v) | u\in{S} \}设 slack(v)=min{lx(u)+ly(v)−w(u,v)∣u∈S}
又∵a=min⁡{lx(u)+ly(v)−w(u,v)∣u∈S,v∈T′}\text{又}\because a = \min \{ lx(u) + ly(v) - w(u,v) | u\in{S} , v\in{T'} \}又∵a=min{lx(u)+ly(v)−w(u,v)∣u∈S,v∈T′}
∴a=min⁡{slack(v)∣v∈T′}\therefore a = \min \{ slack(v) | v\in{T'} \}∴a=min{slack(v)∣v∈T′}

执行更改操作时，遍历 T′T'T′ 中的点 vvv (在 DFS 时顺便进行)，对 lx(u)+ly(v)−w(u,v)lx(u) + ly(v) - w(u,v)lx(u)+ly(v)−w(u,v) 取最小值

DFS 实现

C++
bool dfs(int u)
{
    visx[u] = 1;
    for (int v = 1; v <= n; v++)
    {
        if (w[u][v] == lx[u] + ly[v]) // 在相等子图中
        {
            if (visy[v])
                continue;
            visy[v] = 1;
            if (!match[v] || dfs(match[v]))
            {
                match[v] = u;
                return 1;
            }
        }
        else // 在 T' 中
            slack[v] = std::min(slack[v], lx[u] + ly[v] - w[u][v]);
    }
    return 0;
}

复杂度分析

可以在 O(n)O(n)O(n) 算出顶标修改值 aaa

交错树新增一个点进入 SSS 的时候需要 O(n)O(n)O(n) 更新 slack(v)slack(v)slack(v)。修改顶标需要 O(n)O(n)O(n) 给每个 slack(v)slack(v)slack(v) 减去 a。只要交错树找到一个未匹配点，就找到增广路。

一开始枚举 nnn 个点找增广路，为了找增广路需要延伸 nnn 次交错树，每次延伸需要 nnn 次维护，共 O(n3)O(n^3)O(n3)。

KM 算法

C++
#include <bits/stdc++.h>
using namespace std;
const int N = 510;
const int INF = 0x3f3f3f3f;
int n;                  // 两侧顶点数
int w[N][N];            // 邻接矩阵存储边权
int lx[N], ly[N];       // 顶标数组
int match[N];           // 右部匹配的左部节点
int slack[N];           // 松弛量数组
bool visx[N], visy[N];  // DFS访问标记

bool dfs(int u) {
    visx[u] = true;
    for(int v = 1; v <= n; v++) {
        if(visy[v]) continue;
        int gap = lx[u] + ly[v] - w[u][v];
        if(gap == 0) { // 找到相等子图中的边
            visy[v] = true;
            if(!match[v] || dfs(match[v])) {
                match[v] = u;
                return true;
            }
        } else {
            slack[v] = min(slack[v], gap); // 更新松弛量
        }
    }
    return false;
}

int KM() {
    // 初始化左部顶标
    for(int i = 1; i <= n; i++) {
        lx[i] = -INF;
        for(int j = 1; j <= n; j++)
            lx[i] = max(lx[i], w[i][j]);
    }
    // 为每个左部节点寻找匹配
    for(int i = 1; i <= n; i++) {
        fill(slack+1, slack+n+1, INF);
        while(1) {
            memset(visx, 0, sizeof visx);
            memset(visy, 0, sizeof visy);
            if(dfs(i)) break; // 找到增广路
            // 计算最小调整量a
            int a = INF;
            for(int j = 1; j <= n; j++)
                if(!visy[j]) a = min(a, slack[j]);
            // 调整顶标
            for(int j = 1; j <= n; j++) {
                if(visx[j]) lx[j] -= a;
                if(visy[j]) ly[j] += a;
                else slack[j] -= a; // 维护slack数组
            }
        }
    }
    // 计算总权重
    int res = 0;
    for(int i = 1; i <= n; i++)
        res += w[match[i]][i];
    return res;
}

int main() {
    // 输入处理
    int m; // 实际边数
    cin >> n >> m;
    // 初始化边权（不存在的边设为0）
    memset(w, 0, sizeof w);
    while(m--) {
        int a, b, c;
        cin >> a >> b >> c;
        w[a][b] = c;
    }
    cout << "最大权匹配值: " << KM() << endl;
    return 0;
}
